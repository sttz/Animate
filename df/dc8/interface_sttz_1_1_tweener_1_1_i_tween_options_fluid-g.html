Sttz.Tweener.ITweenOptionsFluid< TContainer > Interface Template Reference

Detailed Description

Fluid interface to set ITweenOptions settings.

Groups and tweens implement a fluid interface to set ITweenOption settings in addition to the properties on ITweenOption itself. This allows to set any number of option in a single instruction:

Animate.Group().Over(5f).Delay(3f).Ease(Easing.Linear);

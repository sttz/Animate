Sttz.Tweener.Plugins.TweenSlerp Class Reference

Detailed Description

Tween a rotation using the shortest path from start to end.

TweenSlerp uses Spherical Linear Interpolation (Slerp) to tween a rotation from start to end using the shortest path possible.

Eg. when not using TweenSlerp, tweening from 10째 to 350째 will take the long way around through 180째, whereas when using TweenSlerp, the tween will go the short way through 0째.

// Explicit usage:
Animate.To(transform, 2f, "rotation", Quaternion.identity, TweenSlerp.Use());
// Automatic usage, plugin auto-detects when it's needed:
Animate.Options.SetAutomatic(TweenSlerp.Automatic());
Animate.To(transform, 2f, "eulerAngles", Vector3.zero);

Sttz.Tweener.ITweenTemplate Interface Reference

Detailed Description

Set of tween options that can be used as a group's defaults.

You can create a tween template using Animate.Template and then use it when creating groups with Animate.On or Animate.Group. The template will override the global options but the group's and tween's options can still override the template's.

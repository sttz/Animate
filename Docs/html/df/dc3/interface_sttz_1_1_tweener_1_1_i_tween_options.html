Sttz.Tweener.ITweenOptions Interface Reference

Detailed Description

Tween options.

Options in Animate can be set in different scopes. If an option is not set in a scope, it will inherit the settings from its parent scope. This allows to define global defaults and override them with various granularity in templates, groups or individual tweens.

The available scopes are:

  • Global defaults at P:Animate.Options
  • Template created using M:Animate.Template (optional)
  • Tween Groups created using M:Animate.On or M:Animate.Group
  • Tweens created using the methods on Animate or Tween
